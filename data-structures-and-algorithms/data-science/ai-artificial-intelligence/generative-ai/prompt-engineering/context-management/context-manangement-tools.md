# Context Management Tools

context-manangement-tools.md


*   OpenWolf

    Rather than compressing shell outputs like RTK, OpenWolf works at the file-read layer. 
    
    It provides the LLM with a file index/token estimate before every read and maintains a learning memory of your 
    corrections across sessions.
    
*   Graphify

    Builds a lightweight local knowledge graph of your codebase so your AI agent can query the structure instead of 
    redundantly reading and re-reading raw files.

# BRD Business Requirement Document

brd-business-requirements-document.md

*   report detailing everything a new project requires for success. 

*   outlines 
    project objectives, 
    what’s expected throughout the project lifecycle
    what’s required to accomplish the project. 

*   7 components

    1.  Executive summary

    2.  Project objectives

    3.  Project scope

    4.  Business requirements

    5.  Key stakeholders

    6.  Project constraints 

    7.  Cost-benefit analysis

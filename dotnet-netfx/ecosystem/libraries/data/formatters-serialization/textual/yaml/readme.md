## YAML

YAML (YAML Ain't Markup Language) is a human-friendly, data-serialization format widely used for configuration files, data storage (e.g., debugging output), and data transmission (e.g., document headers).

It natively supports three fundamental data types:

Scalars - Strings, integers, and floats.
Lists - Ordered sequences of values.
Associative arrays - Key-value mappings.

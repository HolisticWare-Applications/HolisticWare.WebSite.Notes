## HTTP REPL

*   https://docs.microsoft.com/en-us/aspnet/core/web-api/http-repl/

    ```
    dotnet tool install -g \
                Microsoft.dotnet-httprepl
    ```

    ```

    ```

    dotnet tool install -g \



*   MAUI 

*   uses different rendering backends to turn your shared C# or XAML code into native UI on each platform

*   Mobile

    *   Android 
        
        Android widgets 
        
    *   iOS 
    
        UIKit
        
    *   Mac Catalyst
    
        UIKit
        
*   Desktop: Uses WinUI 3 for Windows.Linux & Web (Experimental): The community-driven Avalonia and Gtk4 backends allow MAUI apps to run on WebAssembly, Linux distributions, and older Windows systems (like WPF).

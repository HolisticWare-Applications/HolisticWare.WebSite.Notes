# Worktrees

worktrees.md

*   VS Code

    *   built-in support for Git worktrees, 
    
    *   easy to manage and work with multiple branches at the same time

    *   https://code.visualstudio.com/docs/sourcecontrol/branches-worktrees

*   worktree

    *   separate checkout of a Git branch in its own directory
    
    *   allows user to have multiple working directories for the same repository, each on a different branch. 
    
    *   worktree functionality is especially useful for:

        *   work on multiple features simultaneously in separate folders

        *   Run different versions of your application side by side

        *   Compare implementations across branches

# git tagging

```
git tag SOME_TAG
```

```
git push origin --tags
```

```
git tag -d \
    AndroidX-release-20200214 
git push origin --delete \
    AndroidX-release-20200214

git tag \
    20200214-stable-release \
    967d80032d7cfabcf2079174b5d45a5f2f3da732
git push origin --tags
```
